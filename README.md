# Lumen PHP Framework

##инструкция для проверки

выполнить команду в папке проекта
`cp .env.example .env`

записать свои занчения в .env
`APP_URL`
`DB_*`

выполнить команды 

`composer install`

`php artisan migrate`

`php artisan db:seed`

в выводе команды `db:seed` будет Логин и Пароль менеджера

пароли всех пользователей `secret23`

выполнить команды
`npm install` &&
`npm run prod`
